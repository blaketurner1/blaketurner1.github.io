About This Project

The Font Mannerisms Portfolio is a project that I did as part of Design 3. The goal of this project was to show off a typeface

I chose Helvetica as the typeface that I would research and display. We were tasked with creating a title page, table of contents, family display, specific mannerisms, digital handset type, and an in-context example.

What you see on this website is my project, as well as a contact page with a little bit of information about me.
